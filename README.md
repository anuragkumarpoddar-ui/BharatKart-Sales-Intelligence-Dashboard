# 📊 BharatKart Sales Intelligence Dashboard | Power BI

An end-to-end Business Intelligence project built using **Microsoft Power BI** to analyze and monitor e-commerce sales performance across India.

The dashboard transforms raw sales data into interactive visualizations, enabling businesses to monitor revenue, profitability, customer trends, product performance, and regional sales in real time.

## 📌 Project Overview

This project demonstrates how Power BI can be used to convert transactional sales data into meaningful business insights through interactive dashboards.

The dashboard allows decision-makers to monitor overall business performance, identify high-performing products, evaluate profitability, understand customer purchasing behavior, and analyze sales across different Indian states.

The project was created as a practice Business Intelligence project to strengthen skills in Power BI, data modeling, DAX, visualization design, and business storytelling.

---

## 🎯 Project Objectives

- Analyze online sales performance across India
- Track revenue, profit, and quantity sold
- Monitor sales by state and city
- Identify top-performing product categories
- Compare profitability across products
- Analyze customer purchasing behavior
- Evaluate preferred payment methods
- Build an interactive dashboard for business decision-making

## 📂 Dataset

The project uses two datasets.

### Orders Dataset

Contains customer order information including:

- Order ID
- Order Date
- Customer Name
- State
- City

### Details Dataset

Contains transactional information including:

- Sales Amount
- Profit
- Quantity
- Product Category
- Sub-Category
- Payment Mode
